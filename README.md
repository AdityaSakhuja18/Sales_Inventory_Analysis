# 📊 Sales & Inventory Analysis Dashboard

## 🚀 Project Overview

This project focuses on analyzing product demand and inventory availability using **SQL Server** and **Power BI**. Data from multiple sources was imported into SQL Server, integrated using SQL joins, and transformed into a consolidated table for reporting and visualization.

The dashboard helps monitor inventory levels, identify demand patterns, and support data-driven inventory planning.

## 📂 Data Sources

---

## 🔄 Project Workflow

1. 📥 Imported product and inventory datasets into SQL Server.
2. 🔗 Joined both datasets using **Product_ID**.
3. 🏗️ Created a consolidated table (**dbo.new_table**) for analysis.
4. 📈 Performed SQL-based data validation and aggregation.
5. 🔌 Connected SQL Server data to Power BI.
6. 📊 Built an interactive dashboard for business insights.
